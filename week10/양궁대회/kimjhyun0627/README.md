  # 문제 풀이

## 문제 해설

***bfs***

*해당 과녁에 어피치보다 1발 더 많이 쏘는 경우 or 화살을 아끼고 다른 곳에 더 많이 쏘는 경우*

1. 10점부터 1점까지 모든 화살을 탐색?한다.
2. 해당 과녁에 어피치보다 한 발 더 많이 쏘는 경우와 아예 한 발도 쏘지 않은 경우 두 가지를 모두탐색한다.
3. 화살이 남았다면 마지막 과녁에 모두 쏴주고, 화살이 남지 않았다면 나머지를 continue한다.
4. 해당 탐색의 과녁별 화살 수를 점수차이를 계산하여 max와 비교한 후 res로 반환한다.
5. 예외처리 후 반환한다.


## 시간 복잡도

$$O(len(n)^2)$$
